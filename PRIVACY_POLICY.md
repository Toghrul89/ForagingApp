# Fruit Tree Privacy Policy

Fruit Tree stores foraging logs on the user's device.

Data the app may use

- Location: used when the user chooses to capture GPS coordinates or center the map.
- Camera: used when the user chooses to take a tree photo.
- Photos: used when the user chooses an existing image from the device.
- Local logs: tree name, location text, notes, date, photo URI, fruit type, season, favorite state, and optional coordinates.

Data sharing

The app does not include account login, cloud sync, advertising, analytics, or third-party tracking code. Saved tree logs remain local to the device unless the user later adds export or sharing features.

Map tiles

The map uses an external map tile provider to display map imagery. Opening the map may contact the tile provider from the device.

User control

Users can delete saved logs in the app. Uninstalling the app removes its local app data from the device.

Safety

Fruit Tree is a location journal, not a plant safety or edibility authority. Users should independently confirm plant identity, forage legally, and avoid polluted areas.
